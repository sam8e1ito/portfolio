# Samvel Portfolio

This repository contains my personal portfolio website.

It is a responsive React application where I present my projects, background, tech stack, and contact details. The site also includes a private admin flow for managing portfolio content such as projects and timeline events.

---

## What the website includes

- a home page with hero, about section, tech stack carousel, and latest projects
- a projects page with project cards
- an about page with a vertical timeline
- a contact page with GitHub and email links
- light and dark themes
- a Firebase-backed admin panel for updating content

## Tech stack

- React
- TypeScript
- Vite
- SCSS
- React Router
- Firebase
